# QUIZLY
## Test Your Knowledge the Fun Way
Quizly is an interactive web-based quiz application designed to challenge users on the worldâ€™s most random yet basic facts. With engaging questions sourced through an API and a smooth user experience, Quizly makes learning fun, fast, and refreshing.

Users can choose from three difficulty levels:
- Easy
- Medium
- Hard

Each level contains 10 multiple-choice questions that display one at a time. Score tracking and instant feedback ensure users can evaluate their performance as soon as they finish the quiz.

Quizly is designed to be simple, intuitive, and suitable for anyone who wants to test or improve their general knowledge, whether for fun or learning.
